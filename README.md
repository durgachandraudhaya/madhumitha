# 🤖 Human Following Robot

## 📌 Project Description
This project is an Arduino-based robot that automatically follows a human using ultrasonic and IR sensors. It is designed for applications like luggage carrying, assistance robots, and automation.

## 🚀 Features
- Human detection using ultrasonic sensor
- Obstacle avoidance
- Automatic movement (forward, left, right, stop)
- Simple and efficient design

## 🛠️ Components Used
- Arduino Uno
- Ultrasonic Sensor (HC-SR04)
- IR Sensors
- Motor Driver (L298N)
- DC Motors
- 4-Wheel Robot Chassis (with tires)
- male/female Jumper Wires
- Lithium ion Battery

## ⚙️ Working Principle
The ultrasonic sensor detects the distance between the robot and the human.  
IR sensors help in direction tracking.  
Based on sensor input, the Arduino controls the motors to follow the human.

## 🔌 Circuit Connections (Basic)
- Ultrasonic Sensor → Arduino (Trigger & Echo pins)
- IR Sensors → Arduino digital pins
- Motor Driver → Arduino + Motors
- Power Supply → Lithium ino Battery
